Tu peux maintenant télécharger et installer l’extension Key Idea.

1. Télécharger l’extension

Télécharger Key Idea

2. Installer l’extension

  1. Télécharge le fichier ZIP
  2. Décompresse-le (clic droit → extraire)
  3. Ouvre Google Chrome
  4. Va sur : chrome://extensions
  5. Active le mode développeur (en haut à droite)
  6. Clique sur "Charger l’extension non empaquetée"
  7. Sélectionne le dossier extrait

Une fois installé, tu peux faire clic droit sur un texte → "Save to Key Idea"
